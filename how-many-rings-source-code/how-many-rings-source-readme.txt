HOW MANY RINGS? - SOURCE FILES

This game is built as a single-file browser game.

Files:
- index.html
  Main desktop/responsive version. Includes all HTML, CSS, JavaScript, game data, sounds, and simulator logic.

- how-many-rings-mobile.html
  Mobile-named copy of the same game, useful if you want a separate mobile link.

How to run:
1. Put index.html on any website host.
2. Open it in a browser.
3. No backend, database, or install is required.

Optional:
- Rename index.html to whatever your hosting platform expects.
- Keep how-many-rings-mobile.html only if you want a separate mobile URL.
